
리눅스 셀에서 Base64 인코딩 방법
```
   echo -n "test15@gmail.com:1111" | base64
   토큰 생성  dGVzdDE1QGdtYWlsLmNvbToxMTEx
```


# PowerShell에서 Base64 인코딩 방법
```
$Text = "test15@gmail.com:1111"
$Bytes = [System.Text.Encoding]::UTF8.GetBytes($Text)
$Encoded = [System.Convert]::ToBase64String($Bytes)
$Encoded

```

이미 작성된 파일이 CRLF로 저장되어 있다면, 다음 명령어를 실행해 한 번에 LF로 변환할 수 있습니다.


npx prettier --write .





{
  "singleQuote": true,
  "trailingComma": "all",
  "endOfLine": "lf",

  "rules": {
    "prettier/prettier": [
      "error",
      {
        "endOfLine": "auto"
      }
    ]
  }
}
